# cafe-order-system

Проект демонструє рефакторинг коду системи замовлень у кафе.

Структура
- `CafeBefore.java` — код ДО рефакторингу
- `CafeAfter.java`  — код ПІСЛЯ рефакторингу

Застосовані методи рефакторингу
- Extract Class
- Extract Interface
- Replace Conditional with Polymorphism
- Encapsulate Field
- Move Method
- Extract Method
- Introduce Parameter
